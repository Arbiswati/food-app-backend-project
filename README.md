# Food App Backend Project

This is a beginner-level backend project built using Node.js and Express.

## Features
- User Registration
- User Login (JWT Authentication)
- Get User Profile
- Update User Profile
- Reset & Update Password
- Protected Routes using Auth Middleware

## Tech Stack
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## Project Structure
- controllers/
- routes/
- models/
- middlewares/

## Status
Backend completed. Frontend & deployment coming soon.
